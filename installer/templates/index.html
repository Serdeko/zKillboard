Memcached

Memcached is a cache solution that stores data in memory

APC

APC does the same as Memcached, it is just more tightly integrated with PHP / HHVM, and thus faster

Redis

Redis does the same as APC and Memcached, it is just more fully featured and can do some cool stuff.
